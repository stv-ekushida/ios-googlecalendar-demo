# ios-googlecalendar-demo
iOS Googleカレンダーを利用したデモ(Swift3)

## 準備
### 1.クライアントIDを取得する
[wizard](https://console.developers.google.com/start/api?id=calendar)からクライアントIDを取得する

## 2. PodFileを作成する

```
    pod 'GoogleAPIClient/Calendar', '~> 1.0.2'
    pod 'GTMOAuth2', '~> 1.1.0'
```
